## Descriptions

> 自定义组件，用于展示一个简单的信息列表

### 代码演示

<Playground>
  <Descriptions dataSource={data1} />
</Playground>

<Playground>
  <Descriptions title="豆瓣电影" dataSource={data2} />
</Playground>

## API

### Descriptions

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|title|标题|string|-|
|dataSource|需要展示的信息|Descriptions.Item[]|-|


### Descriptions.Item

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|title|详情标题|string|-|
|content|详情信息|string \| string[]|-|
